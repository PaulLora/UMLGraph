About this program
~~~~~~~~~~~~~~~~~~
NClass is a UML compliant class diagram editor specialized for C# and Java developers.

System requirements
~~~~~~~~~~~~~~~~~~~
- 800 MHz or faster processor
- 128 MB RAM (256 MB recommended)
- 1.0 MB free hard disk space
- Microsoft .NET 2.0 framework
- Windows 2000/XP
- Screen resolution 800x600 or better
